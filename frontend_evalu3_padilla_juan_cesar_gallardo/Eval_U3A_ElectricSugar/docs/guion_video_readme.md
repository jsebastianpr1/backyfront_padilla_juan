# Guion de Video Explicativo: Landing Page - Centro de Negocios Santiago de SERCOTEC

Este guion está diseñado para un video de **4 a 5 minutos** que sirva como presentación técnica y demostración práctica del proyecto, integrando los aspectos del `README.md`, el manual de administración del CMS con Postman/Thunder Client y los aprendizajes de la retrospectiva.

---

## 🎬 Ficha Técnica del Video
* **Duración total estimada:** ~4:30 minutos
* **Tono:** Profesional, técnico, dinámico y resolutivo.
* **Propósito:** Explicar el funcionamiento de la landing page, destacar los componentes React dinámicos, demostrar el CMS simulado con Thunder Client/Postman y mostrar la calidad del flujo de desarrollo git.

## 💡 Consejos Prácticos para la Grabación y Edición

1. **Resolución de Pantalla:** Graba en una resolución estándar de **1080p (1920x1080)** con una relación de aspecto 16:9. Esto evita que el texto del código o las llamadas API se vean borrosos.
2. **Zoom del Editor y Navegador:** En VS Code, aumenta el zoom (con `Ctrl` + `+`) para que las líneas de código del README y de `data.json` sean legibles en pantallas pequeñas. Lo mismo aplica para Postman o Thunder Client.
3. **Fluidez del Scroll:** Cuando muestres el scroll suave de la tarjeta de servicios al formulario de contacto, no muevas el mouse bruscamente. Deja que el callback propio de React haga el desplazamiento automático de forma natural.
4. **Edición de Audio:** Si grabas la voz por separado, utiliza un filtro de reducción de ruido y normaliza el volumen para que tu explicación sea clara, fluida y con un tono entusiasta.
5. **Transiciones:** Utiliza transiciones de corte directo (sin efectos extraños como desvanecimientos a negro muy largos) para pasar de la web al código y a las herramientas API, manteniendo un ritmo ágil y profesional.
